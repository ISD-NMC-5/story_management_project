Giao diện

Dùng nút 🌙/☀️ góc trên để đổi Light / Dark mode. Màu sắc tự lưu theo trình duyệt.

Tìm block trùng

Panel bên trái → Block giống nhau → bấm Quét. Block trùng hiện badge vàng và cảnh báo % giống nhau.

Phím tắt

  • Ctrl + Shift + N Tạo block mới
  • Ctrl + Shift + D Nhân bản block đang chọn
  • Ctrl + Shift + M Gộp block (nhiều block tích chọn hoặc gộp với block sau)

Thao tác nhanh

  • Kéo thả block trong danh sách để đổi thứ tự.
  • Shift + click: chọn dải block liên tiếp (tích xuất).
  • Ctrl + click: chọn/bỏ chọn từng block (tích xuất).
  • Phím tắt không chạy khi đang gõ trong ô nhập liệu.
